## About Will Long

Will Long is the Founder and CEO of [Numinar Analytics](https://www.numinar.com), a political technology startup combining voter data and machine learning to help campaigns win elections. Since its founding in Will's senior year college dorm, Numinar has been deployed on over 2,000 political campaigns and helped reach over 100 million voters.

Before Numinar, he spent time in Silicon Valley at Amazon and Palantir. Will earned a Joint A.B. in Computer Science and Government from Harvard College where he wrote his senior thesis on the application of Thomas Hobbes's political philosophy to problems of cooperation in Artificial Intelligence.

In his spare time, he enjoys discussing philosophy at the Cicero Society, a debate society he founded in Washington, DC, where he lives with his wife Elly.

## ðŸ›  Installation & Set Up

1. Install the Gatsby CLI

   ```sh
   npm install -g gatsby-cli
   ```

2. Install and use the correct version of Node using [NVM](https://github.com/nvm-sh/nvm)

   ```sh
   nvm install
   ```

3. Install dependencies

   ```sh
   yarn
   ```

4. Start the development server

   ```sh
   npm start
   ```

## ðŸš€ Building and Running for Production

1. Generate a full static production build

   ```sh
   npm run build
   ```

1. Preview the site as it will appear once deployed

   ```sh
   npm run serve
   ```
