A business opportunities analytics tool for small businesses built using the U.S. Treasury spending API. This project provides insights into government spending patterns to help small businesses identify potential contract opportunities.

The application features a Flask back-end with a responsive front-end built using Bootstrap, jQuery, and hosted on Heroku. The tool helps small businesses navigate and analyze federal spending data to discover relevant business opportunities.
